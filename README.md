# CS690-FinalProject

The final iteration delivers a modular console application for Daniel's personal expense tracking, budgeting, bill tracking, and critical alerts.

## Documentation

- [Deployment Guide](DEPLOYMENT.md)
- [User Guide](USER_GUIDE.md)

## Run The App

```bash
dotnet run --project FinancialControlApp/FinancialControlApp.csproj
```

## Run Tests

```bash
dotnet test CS690-FinalProject.sln
```

## Implemented Functional Requirements

- `FR1` Manual Transaction Entry
- `FR2` Category Selection
- `FR3` Transaction History List
- `FR4` Spending Summary by Category
- `FR5` Monthly Limit Setting
- `FR6` Budget Progress Indicator
- `FR7` Bill Entry System
- `FR8` Upcoming Bills Ledger
- `FR9` Due Date Flags
- `FR10` Liquidity Warning
- `FR11` Budget Status Indicators
